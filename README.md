## 📌 Project Overview

College Placement Tracker is a custom ServiceNow application designed to streamline campus recruitment by managing student profiles, job postings, and application workflows with automated tracking and analytics.

## ✨ Features

* 👨‍🎓 Student profile management
* 🏢 Company & job posting management
* 📄 Job application tracking
* 🔐 Role-based access control (Student / HR / Admin)
* ⚙️ Automated eligibility validation
* 🔄 Application status workflow (Applied → Shortlisted → Selected)
* 📊 Placement analytics dashboard
* 📩 Notification & workflow automation
* 🧩 Scoped application architecture

## 🛠️ Technologies Used

* ServiceNow Studio
* Scoped Application
* Custom Tables
* Business Rules
* UI Actions
* Flow Designer
* ACL (Access Control)
* Update Sets

## 👥 Roles

* **Student** – Apply for jobs and track status
* **HR** – Post jobs and shortlist candidates
* **Admin** – Monitor placements and generate reports

## 🔄 Workflow

1. Student creates profile
2. HR posts job openings
3. Student applies for eligible jobs
4. HR reviews and shortlists candidates
5. Final selection updated
6. Dashboard reflects placement statistics

## 📊 Outcome

* Simplifies campus recruitment process
* Provides real-time placement tracking
* Improves transparency between students and recruiters
* Demonstrates end-to-end ServiceNow application development
